🤖 Invoice Bot
runs right here.

Drop in one or more PDF invoices and this page rebuilds the core of the Python tool — text extraction, field validation, duplicate detection, and Excel/PDF report generation — entirely client-side.

⚙️ This demo re-implements the extraction & validation engine in JavaScript (via pdf.js) so it can run without a server. Field detection looks for lines like "Invoice #:", "Date:", "Vendor:" and "Total:" — real invoices with different layouts may extract partially, which is exactly what the validator is built to catch. The full project (Python, pdfplumber, batch runs) lives in the repo linked above.
